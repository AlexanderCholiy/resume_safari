Этот сайт — мой личный pet-проект, разработанный с использованием Django. Он представляет собой платформу, где пользователи могут публиковать свои резюме, а также просматривать резюме других пользователей.

Зарегистрированные пользователи могут добавлять, редактировать и удалять свои резюме через удобный REST API. Вся логика API реализована с помощью Django REST Framework, обеспечивая гибкий и масштабируемый подход к управлению данными.

Основной интерфейс сайта ориентирован на просмотр резюме, а взаимодействие с личными данными осуществляется через API. Это позволяет легко интегрировать сайт с внешними frontend-интерфейсами или мобильными приложениями.

Проект создавался с целью практики и углубления знаний в области веб-разработки, а также в работе с REST API и авторизацией пользователей.
